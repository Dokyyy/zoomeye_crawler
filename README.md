# zoomeye_crawler

## **请不要随意使用此脚本，否则责任自负**  

放上来用于学习，使用前需要先申请授权，并自行配置 url。  
```
usage: getip.py [-h] [-e] -d DORK -p PAGES [-f FILENAME]

optional arguments:
  -h, --help            show this help message and exit
  -e, --external        use external zoomeye?
  -d DORK, --dork DORK  dork text
  -p PAGES, --pages PAGES
                        max page to get
  -f FILENAME, --filename FILENAME
                        output file name

```
